## 自己PR
直近では映像配信技術者として3年間経験を積んできました。技術とクリエイティブの架け橋として活動しています。
現在は映像制作会社で動画配信プラットフォーム「DOOONUT」の開発運営に携わり、PMO/QAエンジニア/スクラムマスターとして技術と事業の両面から価値創造に取り組んでいます。
サーバーコスト削減、徹底的なサービス品質の担保、新機能の仕様策定を担当した経験から、技術的な最適化と事業価値の両立を実現できることが私の強みです。

AI技術の急速な進化により「これからのビジネスパーソンには、職種を問わずソフトウェアエンジニアリングの素養が不可欠」という確信を得ました。
この流れは加速して止まることがありません。
私自身、ClaudeやCursorなどのAIツールを活用して業務効率化と品質向上を実感しており、
これを現職の他のメンバー（技術・非技術問わず）にAI推進という形で共有し
雑務から解放され意思決定やUXデザイン、映像のクリエイティブに集中できるようAI推進を実施しております。

「AIと人が共存する社会」という私のモットーは、技術に置き換えられることを恐れるのではなく
「技術を味方につけてより大きな価値を創造」することを意味すると思います。
これからも、技術の進化を楽しみながら人々がより創造的で充実した仕事ができる様貢献して参ります。
